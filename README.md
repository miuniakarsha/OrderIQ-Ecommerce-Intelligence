# OrderIQ: E-commerce Intelligence System

OrderIQ is an end-to-end e-commerce business intelligence and analytics project built using Python, SQL Server, Power BI, Streamlit, and GitHub.

## Project Overview

This project analyzes e-commerce order data to identify customer behavior patterns, delivery performance issues, product-level trends, payment patterns, order lifecycle delays, and business recommendations.

## Main Features

- Customer behavior analysis
- Delivery performance analysis
- Product-level insights
- Payment method analysis
- Order lifecycle analysis
- Time-based trend analysis
- Interactive Power BI dashboard
- Streamlit analytics dashboard
- SQL-based business queries
- Business recommendation insights

## Note

The Streamlit app currently connects to a local SQL Server database. To run it on another computer, the database connection string inside `app/app.py` must be updated.
